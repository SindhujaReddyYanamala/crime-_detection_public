# 🔍 Crime Detection System

## 📌 Project Overview

The Crime Detection System is an AI-based project designed to detect criminal activities such as weapons or suspicious behavior using images/videos. It uses deep learning models for real-time detection.

## 🚀 Features

* Detects weapons (guns, knives, etc.)
* Real-time image/video processing
* Uses trained deep learning models
* Easy to run and extend

---

## 🛠️ Technologies Used

* Python
* OpenCV
* YOLO (Object Detection)
* Machine Learning / Deep Learning


---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/SindhujaReddyYanamala/crime-_detection_public.git
```

2. Navigate to folder:

```
cd crime-_detection_public
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the main file:

```
python main.py
```

---

## ⚠️ Note

* Large dataset and model files are not included due to GitHub size limits.
* You can download datasets separately and place them in the dataset folder.
